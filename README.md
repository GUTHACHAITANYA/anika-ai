# Anika-AI 🖤

### AI-Powered OCR Expense Splitting & Financial Intelligence Platform

Anika-AI is an advanced AI-powered financial assistant designed to simplify group expense management, OCR bill scanning, intelligent expense splitting, and smart financial analytics.

The system combines OCR, NLP, machine learning, and full-stack web technologies to create a smart Splitwise-style financial intelligence platform capable of understanding natural language expenses, analyzing receipts, and automatically calculating settlements between participants.

---

# 🚀 Features

## 🧾 OCR Bill Scanning

* Upload restaurant bills, petrol bills, shopping invoices, medicine receipts, and screenshots
* Multi-bill OCR segmentation
* Automatic item extraction
* Merchant detection
* Tax/GST detection
* PDF/image support

---

## 🤖 AI Expense Intelligence

* Natural language expense understanding
* Automatic name detection
* Amount extraction
* Category classification
* AI-powered settlement calculations

Example:

```text
mohan 500 for petrol
ramu 1200 for food
chaitu 2000 for hotel
```

Automatically converts into structured expense objects.

# 👥 Group Expense Splitting

Supports:

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/GUTHACHAITANYA/anika-ai.git
cd anika-ai
```

---

## 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

## 3️⃣ Run Frontend

```bash
npm run dev
```

---

## 4️⃣ Run Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
